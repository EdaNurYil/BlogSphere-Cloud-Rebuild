## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/EdaNurYil/BlogSphere-Cloud-Rebuild.git
   
   ```

2. **Install dependencies**:
   Ensure you have server.js installed, then run:
   ```bash
   npm install
   ```

3. **Install ejs**:
   Ensure you have server.js installed, then run:
   ```bash
   npm install ejs
   ```

4. **Start the server**:
   Launch the application locally:
   ```bash
   npm start
   ```
